# 환경 변수

> **Refs**: [SECURITY](./SECURITY.md) · [DB](./DB.md)

## 변수 목록
| 변수 | 용도 | 필수 | 예/기본값 |
|---|---|---|---|
| `DATABASE_URL` | DB 연결 | ✅ | `file:./dev.db` |
| `{API_KEY}` | {외부 서비스 인증} | ✅ | — |
| `{OPTION}` | {선택 설정} | ❌ | {기본값} |

- 비밀 값은 절대 커밋하지 않는다. 관리 정책은 [SECURITY](./SECURITY.md) 참조.
- `.env.example`에 키만 두고 값은 비운다.

## LLM 파이프라인 (해당 시)
LLM을 사용하는 백그라운드 파이프라인이 있으면 단계별 입출력과 모델을 명시한다.

| 단계 | 입력 | 처리 | 출력 | 모델 |
|---|---|---|---|---|
| {예: 추출} | {원문} | {프롬프트} | {구조화 데이터} | {모델 id} |
| {예: 요약} | {추출 결과} | {프롬프트} | {요약} | {모델 id} |

- 재시도/타임아웃: {정책}
- 비용·레이트리밋 주의사항: {}
