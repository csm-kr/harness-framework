# 화면 전환 맵

> **Refs**: [SCREENS](./SCREENS.md) · [USER_FLOW](../user/USER_FLOW.md)

화면 간 전환과 조건부 UI 상태를 정의한다. 사용자 관점 단계는 [USER_FLOW](../user/USER_FLOW.md), 화면 상세는 [SCREENS](./SCREENS.md).

## 전환 맵
```
{진입점}
   │
   ├─▶ {화면 A} ──{액션}──▶ {화면 B}
   │                          │
   │                          └──{조건}──▶ {화면 C}
   │
   └─▶ {화면 D}
```

## 화면별 진입/이탈
| 화면 | 진입 경로 | 이탈 경로 |
|---|---|---|
| {화면 A} | {어디서} | {어디로} |
| {화면 B} | {어디서} | {어디로} |

## 조건부 UI 상태
| 화면 | 조건 | 표시 |
|---|---|---|
| {화면} | {미인증} | {로그인 유도} |
| {화면} | {데이터 없음} | {빈 상태} |
| {화면} | {로딩 중} | {스켈레톤} |
| {화면} | {에러} | {에러 + 재시도} |
